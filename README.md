# Magisk SELinux Manager
A Magisk Module dedicated to the manipulation and analysis of SELinux

## Features
  * Set SELinux Mode (Enforcing or Permissive) at startup

## Dependencies
  * Magisk v17.0+
  * Android v4.3+ device
  * SELinux Enabled Kernel
